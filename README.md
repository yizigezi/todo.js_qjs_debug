## 快速上手

### 1. 开发

```
npm install
npm run start
```

### 2. 构建

```
npm run build
npm run release
```

### 3. 调试

```
npm run watch
```

代码规范化可以帮助开发者在git commit前进行代码校验、格式化、commit信息校验
